### What I learned

I learnt how to use CSS grid to position elements, also I relearnt the use of proper CSS units.

```css
.features .wrapper {
    display: grid;
    grid-template-columns: repeat(2, minmax(auto, 400px));
    gap: 100px 100px;
    justify-content: center;
}
```
